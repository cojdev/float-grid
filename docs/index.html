Float Grid

A simple Float-based CSS Grid System built with Sass.

View on Github

Global Variables

$columns Total number of columns.
$gutter Space between columns.

Mixins

Row row(size, gutter)
Column column(size, gutter, min-break, max-break)

Column Sizes

Columns are a sized based on the $columns value (default is 12).

col-4

col-8

<div class="row">
    <div class="col-4">
        <p>col-4</p> 
    </div>
    <div class="col-8">
        <p>col-8</p> 
    </div>
</div>
.col-4 {
    @include column(4);
}

.col-8 {
    @include column(8);
}
